Count Frequency of Each Element
Question:
Given an array, count how many times each element appears.
Constraints:
1 £ N £ 10^5
-10^9 £ arr[i] £ 10^9
Sample Input 1:
[1,2,2,3,1]
Sample Output 1:
{1:2, 2:2, 3:1}
Explanation:
1 appears twice, 2 appears twice, 3 appears once.
Sample Input 2:
[4,4,4,2]
Sample Output 2:
{4:3, 2:1}
Explanation:
4 appears three times, 2 appears once.

First Repeating Element
Question:
Return the first element that appears more than once.
Constraints:
1 £ N £ 10^5
Sample Input 1:
[10,5,3,4,3,5,6]
Sample Output 1:
5
Explanation:
5 is the first element whose frequency is greater than 1.
Sample Input 2:
[1,2,3,4]
Sample Output 2:
-1
Explanation:
No element repeats.
